# 可灵分镜提示词规范

## 1. 目标

这份规范用于把短剧分镜表转换成适合可灵视频生成的提示词。它不是普通画面描述，而是面向“可灵文生视频 / 图生视频 / 首尾帧 / 多镜头 / 主体一致性”的生产提示词。

核心目标：

- 让单条分镜能稳定生成 5-10 秒视频。
- 让角色、场景、动作、光影和镜头运动表达清楚。
- 降低可灵生成时的跑偏、动作过载、角色漂移和镜头混乱。
- 方便后续接入可灵 API 或人工复制到可灵工作台。

---

## 2. 官方规则提炼

可灵官方提示词公式：

```text
主体（主体描述） + 运动 + 场景（场景描述） + 镜头语言 + 光影 + 氛围
```

结合短剧生产，需要扩展为：

```text
主体身份与外观
+ 当前动作
+ 场景空间
+ 镜头景别
+ 运镜方式
+ 光影色调
+ 情绪氛围
+ 台词 / 旁白
+ 负面约束
```

---

## 3. 可灵短剧分镜提示词结构

### 标准模板

```text
【主体】
{角色/主体名称}，{年龄/身份/服装/状态/表情}。

【动作】
{一个连续动作链，不超过 2-3 个动作节点}。

【场景】
{地点}，{前景/中景/背景空间关系}，{关键道具或环境元素}。

【镜头】
{景别}，{机位}，{运镜}，{是否一镜到底/是否切换镜头}。

【光影与氛围】
{主光源方向}，{冷暖色调}，{明暗关系}，{情绪氛围}。

【声音/台词】
{角色名}（{语气}）：“{台词}”
或：无台词，仅保留环境音：{环境音}

【生成约束】
保持角色五官、服装和发型一致；动作自然；不要新增人物；不要改变场景；不要文字乱码；不要肢体畸形。
```

---

## 4. 可灵友好的写法原则

### 4.1 单镜头少做事

一条 5 秒分镜只安排一个核心动作。

推荐：

```text
苏婉缓缓抬头，眼神从恐惧转为警觉，右手攥紧旧信。
```

不推荐：

```text
苏婉醒来、奔跑、找到证据、和反派争吵、最后跳窗逃走。
```

### 4.2 动作要连续，不要抽象

推荐：

```text
她低头看向旧信，指尖发抖，随后猛地抬眼看向门口。
```

不推荐：

```text
她非常害怕但又很坚定。
```

### 4.3 场景要有空间层次

推荐：

```text
前景是模糊的红色幕布，中景是苏婉半跪在舞台中央，背景是破旧剧院座椅和闪烁的安全出口灯。
```

不推荐：

```text
一个废弃剧院，很阴森。
```

### 4.4 镜头语言要明确

推荐：

```text
近景，低机位，镜头缓慢推近她的脸，焦点从旧信转移到她的眼睛。
```

不推荐：

```text
电影感镜头，氛围好。
```

### 4.5 可灵多镜头提示要分段

如果使用可灵 3.0 多镜头能力，可以写成：

```text
第一镜：中景，苏婉在废弃剧院醒来，镜头缓慢推近。
第二镜：特写，旧信上的名字被雨水打湿。
第三镜：近景，苏婉抬头看向门口，听见妹妹的声音。
```

但每个镜头都要保持动作简单，不要把完整剧情塞进一个镜头。

---

## 5. 负面提示词模板

```text
不要新增人物，不要改变角色服装，不要改变发型，不要肢体畸形，不要多手指，不要脸部变形，不要镜头乱跳，不要画面闪烁，不要文字乱码，不要低清晰度，不要卡通化，不要过度磨皮，不要无关字幕，不要水印。
```

如果是图生视频或首尾帧：

```text
严格保持起始图中角色五官、服装、发型、场景结构和光影方向一致，不要替换主体，不要改变画面时代背景。
```

## 6. 短剧分镜示例

### 示例 1：悬疑开场

```text
【主体】
苏婉，二十多岁女性，黑色风衣，头发微湿，脸色苍白，手里攥着一封旧信。

【动作】
她从废弃剧院舞台中央缓缓坐起，低头看向旧信，指尖发抖，随后猛地抬眼看向观众席深处。

【场景】
废弃剧院，前景是模糊的红色幕布，中景是苏婉，背景是破旧座椅和闪烁的安全出口灯。

【镜头】
近景，低机位，镜头从旧信缓慢推近到苏婉的脸，焦点从纸面转移到她的眼睛。

【光影与氛围】
左侧冷蓝色弱光，背景大面积暗部，空气中有细小灰尘，压抑、悬疑、危险感。

【声音/台词】
无台词，仅有雨声、远处电流滋滋声和轻微呼吸声。

【生成约束】
保持角色五官和服装一致；动作自然；不要新增人物；不要改变剧院场景；不要文字乱码；不要肢体畸形。
```

### 示例 2：强钩子台词

```text
【主体】
苏婉，黑色风衣，眼眶发红但强忍情绪，手持手机。

【动作】
她盯着手机屏幕，呼吸停住，慢慢后退半步，手指用力攥紧手机。

【场景】
废弃剧院后台，前景是半开的铁门，中景是苏婉，背景是旧化妆镜和闪烁灯泡。

【镜头】
特写，固定镜头，浅景深，手机屏幕虚化在前景，苏婉的眼睛逐渐对焦。

【光影与氛围】
顶部冷白灯闪烁，面部一半明一半暗，紧张、惊疑、背叛感。

【声音/台词】
手机里妹妹的声音（虚弱、断续）：“姐姐，别相信你身边的人。”

【生成约束】
保持角色一致；嘴型自然；不要新增其他人；不要改变手机位置；不要字幕乱码；不要脸部变形。
```

### 示例 3：图生视频 / 首帧模式

```text
基于首帧图生成 5 秒视频。严格保持首帧中的人物五官、黑色风衣、湿发状态、废弃剧院背景和冷蓝色光影。

苏婉站在舞台中央，低头看着手中的旧信。镜头从中景缓慢推近到近景，她的手指轻微颤抖，随后抬头看向镜头右侧，眼神从迷茫变为警觉。背景安全出口灯轻微闪烁，空气中有灰尘漂浮。整体氛围压抑、悬疑、危险。

负面提示词：不要改变人物长相，不要改变服装，不要新增人物，不要改变场景结构，不要肢体畸形，不要画面闪烁，不要文字乱码，不要水印。
```

## 7. 给系统使用的转换规则

从分镜表转可灵提示词时，每条分镜读取以下字段：

| 分镜字段 | 转成可灵提示词 |
|---|---|
| 画面描述 | 主体 + 场景 + 构图 |
| 角色动作 | 动作 |
| 景别 | 镜头 |
| 运镜 | 镜头 |
| 情绪 | 氛围 |
| 光影氛围 | 光影 |
| 台词 | 声音/台词 |
| 关联资产 | 主体一致性约束 |

转换时必须：

1. 每条分镜只保留一个核心动作。
2. 明确主体、场景、镜头、光影、氛围。
3. 台词必须标明角色名和语气。
4. 如果有首帧图，优先写“严格保持首帧中的主体和场景一致”。
5. 自动补充负面提示词。
